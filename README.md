##  Funcionalidades

- **Menu Mobile**: Navegação otimizada para dispositivos móveis
- **Links Ativos**: Destaque automático da seção atual
- **Formulário com API**: Envio de mensagens via FormSubmit
- **Animação de Scroll**: Efeitos visuais ao rolar a página
- **Alternar Modo Claro/Escuro**: Troca de tema
- **Animações da Seção**: Transições entre seções
- **Botão Voltar ao Topo**: Navegação rápida para o início
- **Carrossel de Projetos**: Apresentação idos trabalhos

##  Tecnologias Utilizadas

### Frontend
- **HTML5**: Estrutura semântica
- **CSS3**: Estilização responsiva e animações
- **JavaScript**: Interatividade e funcionalidades dinâmicas

- **Início**: Apresentação pessoal
